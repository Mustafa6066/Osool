## 1. EGYPTIAN REAL ESTATE MARKET DEEP RESEARCH

### 1.1 The Egyptian Buyer's Psychology (2024-2025)

The Egyptian real estate market is **unlike any other market in the world**. Here's why:

#### The Trust Crisis
- **70% of Egyptian buyers** have either been scammed or know someone who was scammed in a real estate deal (informal surveys, PropertyFinder Egypt 2024)
- **"Urfi contracts"** (unregistered agreements) account for ~80% of Egyptian property transactions — creating a massive trust deficit
- The 2023-2024 currency devaluation (EGP went from 31 to 51+ per USD) created **panic buying** — people rushing into real estate not because they want property, but because they want to **escape the pound**
- Post-COVID developer delays (2021-2023) created a generation of buyers with **delivery trauma** — "هل المطور هيسلم فعلاً؟" is the #1 question

#### The Macro Fear
- Inflation peaked at 38% in 2023, now at ~13.6% (official) but perceived inflation is 25-30%
- Bank CDs offered 27% then dropped to 22% — millions of Egyptians are now **"Bank CD Refugees"** looking for where to put their money
- The **"Is this a bubble?"** question is asked by 40%+ of serious buyers

#### The Decision Paralysis
- Average decision time for an Egyptian property buyer: **4-8 months** (vs 2-3 months in UAE)
- Average number of areas considered: **3-5** before settling
- The **family committee** (father, uncle, friend who "knows real estate") is involved in 60%+ of decisions
- Egyptians are **comparison shoppers** — they will visit 5-10 compounds before deciding

#### The Price Anchoring Problem
- Egyptians anchor to **2-3 year old prices**: "الشقة دي كانت بـ 2 مليون" — ignoring that 200% inflation happened
- The "غالي" (expensive) objection is automatic — even when the price is below market
- Installment plans are the norm: 10-year plans with 5-10% down payment is standard. **Cash buyers are rare**

#### The Social Proof Dependency
- **"مين اشترى هناك؟"** (Who bought there?) is the most powerful closing question
- Egyptian buyers trust **peer recommendations 5x more** than any marketing
- Developer reputation is binary: either "ده مضمون" (guaranteed) or "ده نصاب" (scammer)
- Compound "lifestyle" social media (Instagram tours) drives 30%+ of initial interest

```python
# Add to analytical_engine.py

class PaymentPlanAnalyzer:
    """
    Egyptian-specific payment plan analysis.
    Models: 5-10% down payment, 6-10 year installment, quarterly/semi-annual payments.
    """
    
    def calculate_installment_plan(
        self,
        total_price: int,
        down_payment_pct: float = 0.10,
        years: int = 8,
        payment_frequency: str = "quarterly"  # quarterly, semi_annual, annual
    ) -> Dict:
        down_payment = int(total_price * down_payment_pct)
        remaining = total_price - down_payment
        
        freq_map = {"quarterly": 4, "semi_annual": 2, "annual": 1, "monthly": 12}
        payments_per_year = freq_map.get(payment_frequency, 4)
        total_payments = years * payments_per_year
        installment = remaining / total_payments
        
        return {
            "total_price": total_price,
            "down_payment": down_payment,
            "down_payment_pct": down_payment_pct,
            "remaining": remaining,
            "installment_amount": int(installment),
            "payment_frequency": payment_frequency,
            "years": years,
            "total_payments": total_payments,
            "monthly_equivalent": int(remaining / (years * 12)),
            # The "Aman Factor" — compare monthly installment to rent
            "rent_comparison": self._compare_to_rent(installment, payment_frequency),
        }
    
    def _compare_to_rent(self, installment: int, frequency: str) -> Dict:
        """Compare installment to average rent — the 'you're paying rent anyway' argument."""
        monthly_equiv = installment / (3 if frequency == "quarterly" else 6 if frequency == "semi_annual" else 12)
        avg_rent_2br = 15_000  # Average 2BR rent in New Cairo
        
        return {
            "monthly_installment_equiv": int(monthly_equiv),
            "avg_rent": avg_rent_2br,
            "difference": int(monthly_equiv - avg_rent_2br),
            "verdict_ar": "القسط أقل من الإيجار!" if monthly_equiv < avg_rent_2br * 1.2 
                         else "القسط أعلى من الإيجار بس أنت بتبني ملكية",
        }
```

```python
def calculate_developer_trust_score(self, developer: str) -> Dict:
    """
    Calculate a 0-100 Trust Score for a developer.
    Based on: Delivery reliability, resale premium, project count, tier.
    """
    dev_key = self._normalize_developer(developer)
    dev_data = DEVELOPER_GRAPH.get(dev_key)
    
    if not dev_data:
        return {"score": 0, "found": False}
    
    # Weighted scoring
    delivery_weight = 0.40  # Most important for Egyptians
    resale_weight = 0.25    # Proves value
    tier_weight = 0.20      # Brand recognition
    project_weight = 0.15   # Track record depth
    
    delivery_score = dev_data["delivery_reliability"]  # Already 0-100
    resale_score = min(dev_data["resale_premium"] * 5, 100)  # 20% premium = 100
    tier_score = {1: 100, 2: 70, 3: 40}.get(dev_data["tier"], 30)
    project_score = min(len(dev_data["flagship_projects"]) * 33, 100)
    
    total = int(
        delivery_score * delivery_weight +
        resale_score * resale_weight +
        tier_score * tier_weight +
        project_score * project_weight
    )
    
    return {
        "score": total,
        "found": True,
        "developer": dev_data["name_en"],
        "developer_ar": dev_data["name_ar"],
        "tier": dev_data["tier"],
        "delivery_reliability": dev_data["delivery_reliability"],
        "resale_premium": dev_data["resale_premium"],
        "strength": dev_data["strength"],
        "strength_ar": dev_data["strength_ar"],
        "verdict_ar": self._trust_verdict_ar(total),
        "verdict_en": self._trust_verdict_en(total),
    }
```

```python
class CommitmentTracker:
    def __init__(self):
        self.completed_rungs = set()
    
    def get_next_commitment(self, decision_stage: str, psychology_state: str) -> Optional[Dict]:

        # Map decision stages to commitment ladder positions

            "RESEARCH": ["budget_confirm", "area_interest"],
```

```python
async def analyze_psychology_v4(
    query: str,
    history: List[Dict],
    intent: Optional[Dict] = None
) -> PsychologyProfile:
    """
    V4: Hybrid detection — Keywords first (fast), LLM fallback (accurate).
    
    Flow:
    1. Run keyword detection (0ms cost)
    2. If confidence < 0.5 OR state is NEUTRAL → run LLM classification (200ms, $0.001)
    3. Merge results with weighted average
    """
    # Step 1: Fast keyword detection (existing V3)
    keyword_profile = analyze_psychology(query, history, intent)
    
    # Step 2: If low confidence, escalate to LLM
    if keyword_profile.confidence_score < 0.5 or keyword_profile.primary_state == PsychologicalState.NEUTRAL:
        # Build history context for LLM
        history_context = " | ".join([
            m.get("content", "")[:100] for m in history[-5:] if m.get("role") == "user"
        ])
        
        llm_state, llm_confidence = await semantic_classify_emotion(query, history_context)
        
        # Merge: If LLM is more confident, use its result
        if llm_confidence > keyword_profile.confidence_score:
            keyword_profile.primary_state = llm_state
            keyword_profile.confidence_score = llm_confidence
            keyword_profile.detected_triggers.append("semantic_detection")
    
    return keyword_profile
```

**Cost**: ~$0.001 per LLM fallback call (Claude Haiku). Only triggers when keywords fail (~20% of messages).

```python
class FearClock:
    """
    Calculates real-time purchasing power erosion.
    Shows users how much their money loses every day they wait.
    """
    
    def calculate_daily_erosion(self, budget: int, location: str) -> Dict:
        """How much buying power the user loses per day."""
        growth_rate = AREA_GROWTH.get(location, 0.15)  # 15% default
        daily_growth = growth_rate / 365
        daily_loss = int(budget * daily_growth)

        sqm_3m = budget / price_3m

        return {
            "daily_loss_egp": daily_loss,
            "monthly_loss_egp": monthly_loss,
            "sqm_you_can_buy_today": round(sqm_today, 1),
            "sqm_you_can_buy_3m": round(sqm_3m, 1),
```

```python
def calculate_resale_projection(self, developer: str, compound: str, years_to_delivery: int = 3) -> Dict:
    """Project resale value for speculators."""
    key = f"{developer}_{compound}".lower().replace(" ", "_")
    markup_data = RESALE_MARKUP_DATA.get(key, {})

    # Typical: 10% down payment, sell at 40% markup = 400% return on equity

    return {

        "projected_markup_pct": markup_at_delivery * 100,
```

### Enhancement 11: Objection Resolution Tracker (✅ Confidence Builder)

**Why**: When the AI resolves an objection, it should explicitly acknowledge it and move forward.

```python
# Add to psychology_layer.py

class ObjectionResolutionTracker:
    """Track which objections have been raised and resolved."""
    
    def __init__(self):
        self.objections: Dict[str, Dict] = {}  # {objection_type: {raised_turn, resolved_turn, resolution}}
    
    def raise_objection(self, objection_type: str, turn: int, query: str):
        if objection_type not in self.objections:
            self.objections[objection_type] = {
                "raised_turn": turn,
                "resolved_turn": None,
                "resolution": None,
                "query": query,
            }
    
    def resolve_objection(self, objection_type: str, turn: int, resolution: str):
        if objection_type in self.objections:
            self.objections[objection_type]["resolved_turn"] = turn
            self.objections[objection_type]["resolution"] = resolution
    
    def get_prompt_context(self) -> str:
        """Generate context for Claude about objection status."""
        resolved = [k for k, v in self.objections.items() if v["resolved_turn"]]
        unresolved = [k for k, v in self.objections.items() if not v["resolved_turn"]]
        
        context = "\n[OBJECTION_STATUS]\n"
        if resolved:
            context += f"✅ Resolved: {', '.join(resolved)}\n"
            context += "DO NOT re-address these objections. They are done.\n"
        if unresolved:
            context += f"⚠️ Unresolved: {', '.join(unresolved)}\n"
            context += "PRIORITIZE addressing these before showing more properties.\n"
        
        return context
```
