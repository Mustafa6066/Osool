# Osool - Claude AI-Inspired UI

This project features a **Claude AI-inspired chat interface** with warm beige aesthetics and centered layout.

## Recent Updates (Jan 2026)

### ✨ Claude-Inspired UI Redesign
- Warm beige background (#F5F3EF)
- Burnt orange accent (#D97757)  
- Centered 672px layout for focused reading
- Substantial multi-line input box
- Serif font headings (Merriweather)
- Integrated paperclip + send buttons

## Quick Start

```bash
# Install dependencies
cd web
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

Visit `http://localhost:3000/chat` to see the new UI in action!

## Deployment

The project auto-deploys to Vercel on push to master.
